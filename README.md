# 🛒 Blinkit Sales Data Analysis

## 📌 Objective
Analyze Blinkit sales data to uncover patterns in customer behavior, product performance, and outlet efficiency.

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Key Analysis
- Sales by Item Type
- Fat Content Analysis
- Outlet Size & Location Performance
- Data Cleaning & Visualization

## 🔍 Key Insights
- Tier 3 locations generate higher sales
- Medium-sized outlets perform best
- Low-fat products are more preferred

## 💡 Business Recommendations
- Focus expansion in Tier 3 areas
- Increase stock of high-performing items
- Optimize medium outlets

- Blinkit_EDA_Project.ipynb → Main notebook
- blinkit_data.csv → Dataset
